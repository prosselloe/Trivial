# Trivial de la CiberSeguretat

Benvingut/da al **Trivial de la CiberSeguretat**, una aplicació desenvolupada amb Flutter i dissenyada per posar a prova els teus coneixements sobre seguretat informàtica.

## 🚀 Sobre el Joc

El Trivial de la CiberSeguretat és un joc de preguntes i respostes per a 1 a 4 jugadors. L'objectiu és simple: llença el dau, mou la teva fitxa pel tauler i respon correctament les preguntes de les diferents categories per aconseguir tots els formatgets.

### 🎲 Característiques Principals

- **Partides de 1 a 4 jugadors:** Juga sol o amb amics.
- **Tauler interactiu:** Mou la teva fitxa pel tauler i visualitza la teva posició i la dels teus contrincants.
- **Dau virtual:** Llença el dau per determinar quantes caselles et mous.
- **6 Categories de preguntes sobre ciberseguretat:**
  - 🔵 **Ordinadors:** Maquinari, programari i conceptes bàsics.
  - 🟢 **Internet:** Navegació, xarxes socials i comunicació en línia.
  - 🔴 **Mòbils i tauletes:** Sistemes operatius, aplicacions i seguretat.
  - 🟡 **Llocs web i compres:** Transaccions segures i protecció de dades.
  - 🟠 **Ciberseguretat:** Atacs, prevenció i bones pràctiques.
  - 🟣 **Conceptes Generals:** Altres temes clau de seguretat digital.
- **Formatgets:** Aconsegueix un formatget de cada categoria responent correctament a les caselles especials.
- **Sistema de torns:** El joc gestiona automàticament els torns dels jugadors.
- **Puntuació:** Visualitza la teva puntuació i els formatgets que has aconseguit.

## 🛠️ Desenvolupat amb

- **Flutter:** Un framework de codi obert per a crear aplicacions multiplataforma d'alta qualitat.
- **Dart:** El llenguatge de programació utilitzat per Flutter.
- **Provider:** Per a la gestió de l'estat de l'aplicació.
- **GoRouter:** Per a la gestió de la navegació entre pantalles.

## 🏁 Com començar

Per a executar aquest projecte, necessitaràs tenir Flutter instal·lat al teu ordinador. Un cop el tinguis, segueix aquests passos:

1. **Clona el repositori:**
   ```bash
   git clone <URL_DEL_REPOSITORI>
   ```
2. **Navega al directori del projecte:**
   ```bash
   cd trivial_ciberseguretat
   ```
3. **Instal·la les dependències:**
   ```bash
   flutter pub get
   ```
4. **Executa l'aplicació:**
   ```bash
   flutter run
   ```
